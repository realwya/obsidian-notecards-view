# Obsidian Notecards View Plugin

A plugin for Obsidian that adds a new base view type, "Notecards View", which displays your notes as a grid of cards, providing a visual and organized way to browse your markdown files.
![SCR-20260130-jpzf](https://github.com/user-attachments/assets/0ccc2ec2-1615-4749-9354-ee206fe68421)



## Features

- **Card-based View**: Displays your notes as a grid of cards

## Installation

### Manual Installation

1. Download the latest release from the GitHub repository
2. Extract the files
3. Open your vault's plugin folder: `VaultFolder/.obsidian/plugins/`
4. Create a new folder named `notecards-view`
5. Copy the 'manifest.json', 'styles.css', 'main.js' into the `notecards-view` folder
6. Restart Obsidian
7. Enable the plugin in Settings â†’ Community Plugins
8. In your base file, click 'Add view' and select 'Notecards' in Layout dropdown menu.
